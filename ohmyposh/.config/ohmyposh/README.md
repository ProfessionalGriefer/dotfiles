# Oh My Posh

```sh
# Export default config
oh-my-posh config export --format json --output ~/.config/ohmyposh/base.json 
oh-my-posh config export --format toml --output ~/.config/ohmyposh/macchiato.toml 

# In Fish config file
oh-my-posh init fish --config ~/.config/ohmyposh/catppuccin_macchiato.json | source
```
